# 網域名稱系統 Domain Name System(DNS)

網域名稱系統(Domain Name System(DNS))是保存並提供第三方查詢之域名-IP 位址對應關係的分散式資料庫系統，讓使用者可以以好記的名稱而非 IP 地址訪問各種網路資源
